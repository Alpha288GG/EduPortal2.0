Mobile Preview Guide

How to Preview Materials on Mobile

Our educational portal now offers a mobile preview feature that allows you to view study materials directly on your mobile device without downloading them first. This guide will walk you through how to use this feature.

1 Access the Materials Page

Navigate to the "Study Materials" section from the dashboard or main menu.

2 Find Your Material

Use the filters to select the academic year, semester, subject, and category to find the material you want to preview.

3 Use the Preview Button

Each material card has a "Preview" button (with an eye icon) that appears on mobile devices. Tap this button to open the material in the preview mode.

4 Interact with the Preview

Once the preview opens, you can:

  • Scroll through document content
  • Zoom in/out using pinch gestures (for images and PDFs)
  • Close the preview by tapping the X in the top-right corner
  • Download the file if needed using the download button

Tip

For the best experience, use the preview feature with a stable internet connection. Some file types may not be previewable and will offer a download option instead.

Device Compatibility

The preview feature works best on modern smartphones and tablets with updated browsers. If you encounter any issues, try updating your browser or switching to Chrome, Safari, or Firefox.

Supported File Types

The following file types can be previewed directly:

  • PDF documents
  • Images (JPG, PNG, GIF)
  • Text files
  • HTML and code files

Other file types (like Word documents, PowerPoint presentations) may require downloading first, depending on your device's capabilities.
